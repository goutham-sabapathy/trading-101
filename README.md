# trading-101

A shared, vendor-neutral knowledge base for general trading education and best
practices — strategy mechanics, indicators, risk and discipline frameworks, and
rule-based decision systems.

> Educational reference only. Nothing here is financial advice. Validate every
> rule and threshold with your own backtesting and paper trading before risking
> capital.

## Where to start

- **[Documentation home](./docs/README.md)** — scope and folder layout.
- **[Common Knowledge Base Index](./docs/common/README.md)** — the categorized
  index of every concept doc, with one-line descriptions and reading paths.
- **[Options Strategy Builder](./docs/strategy-builder/index.html)** — an in-browser
  payoff/Greeks visualizer (open the file in a browser).

## Repository layout

```text
docs/
  README.md            # documentation home
  common/              # the knowledge base (concepts, playbooks, frameworks)
  strategy-builder/    # interactive options payoff/Greeks visualizer
```

## Contributing

Keep this repo generic and educational. Do not store project-specific strategy
ideas, private drafts, or proprietary execution plans here. When you add a doc to
`docs/common/`, link it from `docs/common/README.md` under the appropriate category.
