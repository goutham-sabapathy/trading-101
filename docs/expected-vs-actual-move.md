# Expected Move vs Actual Move Analysis

This is the core of earnings edge detection.

---

# How To Calculate Expected Move

ATM Call Premium + ATM Put Premium

Example:
Call = $8
Put = $7
Expected move = $15

---

# Compare With Actual Move

If stock moves $10:
→ Premium overpriced
→ Selling favored

If stock moves $20:
→ Premium underpriced
→ Buying favored

---

# What To Track

For last 8–12 earnings:

- Expected move %
- Actual move %
- Difference
- IV before
- IV after
- IV crush %

This becomes your statistical profile.
