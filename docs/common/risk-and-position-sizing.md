# Risk and Position Sizing (Generic)

## Trade-Level Controls

- Define max loss per position before entry.
- Cap position risk as a fixed portfolio percentage.
- Avoid illiquid contracts with wide spreads.
- Use defined-risk structures when short volatility.

## Portfolio-Level Controls

- Limit concurrent event exposure.
- Cap sector concentration.
- Monitor aggregate delta/vega/gamma.
- Use volatility-regime kill-switch rules.

## Process Controls

- No strategy without explicit exit rules.
- No manual overrides without documented reason code.
- Log every recommendation change with trigger context.
