# Trading Journal Template

## Template

```text
Date:
Market Bias:
Setup Name:
Underlying:
Option (Strike / Expiry):

Entry Reason:
Stop:
Target:

Risk (R):
Result (R):

Rules Followed? (Y/N)
If No, why?

Emotion Before:
Emotion During:
Emotion After:

One Improvement:
```

## Usage Notes

- Record each trade immediately after exit.
- Use consistent labels for setup names.
- Weekly review should focus on rule adherence before P&L.
